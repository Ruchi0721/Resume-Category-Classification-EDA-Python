# Resume-Category-Classification-EDA-Python
Manually screening hundreds of resumes is time-consuming for recruiters. This project automates that process by building a multi-class text classifier that reads a raw resume and predicts the most relevant job category.

# Key highlights:

•	Cleans raw resume text (removes URLs, hashtags, special characters, emojis)

•	Handles class imbalance via oversampling

•	Converts text to numerical features using TF-IDF

•	Trains and compares KNN and SVM classifiers

•	Provides a ready-to-use pred() function for real-world resume prediction
